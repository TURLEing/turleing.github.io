壹. 拉格朗日插值：

1. 插值：

给定一系列点（共 n+1 个），确定一个 n 次多项式使得这些点在该函数图像上。
In other words, 就是将 点值表示 转换为 系数表示 的过程。

2. 拉插：

实质是对于每个点 (xi,yi)(x_i,y_i)(xi​,yi​) 都凑一个式子，在联立求出所有的系数。
那怎么凑这个式子呢？由多项式 余数和商 的定义，我们不难给出一个引理：

f(x)≡f(xi)≡yi (mod (x−xi))f(x)\equiv f(x_i)\equiv y_i~(mod~(x-x_i)) f(x)≡f(xi​)≡yi​ (mod (x−xi​))

上式可以作差证明。对每个点都列出该式，再套上 中国剩余定理 的公式，我们便可以得到拉插的一般形式：

f(x)=∑i=1n( yi⋅∏i≠jx−xjxi−xj )(mod M)f(x)=\sum_{i=1}^n (~y_i \cdot\prod_{i\neq j} \frac{x-x_j}{x_i-x_j}~) (mod~ M) f(x)=i=1∑n​( yi​⋅i​=j∏​xi​−xj​x−xj​​ )(mod M)

貳.快速傅里叶变换：

1. 概念：

给定两个不高于 n 次的多项式 f(x),g(x)f(x),g(x)f(x),g(x)，可以在 O(n⋅logn)O(n\cdot logn)O(n⋅logn) 内求出 f(x)⋅g(x)f(x)\cdot g(x)f(x)⋅g(x) 各个系数的算法。

其实质是利用 复数单位根 的种种性质，将两函数的系数表示转化为点值表示，在通过 逆变换 求出新函数的点值表示.

2. DFT 的实现：

对于两函数的系数表示，利用复数单位根性质将 ωn1,ωn2...ωnn\omega_n^1,\omega_n^2...\omega_n^nωn1​,ωn2​...ωnn​ 带入，进行奇偶项分治，并最终求出整个函数对应点值的过程。

设 F(x)F(x)F(x) 为当前分治总函数的点值序列，O(x)O(x)O(x) 为奇数项分治后的序列，E(x)E(x)E(x) 为偶数项分治后的序列，根据性质：

F(ωnk)=O(ωn/2k)+ωnk×E(ωn/2k)F(ω^k_n)=O(ω^k_{n/2})+ω^k_n\times E(ω^k_{n/2}) F(ωnk​)=O(ωn/2k​)+ωnk​×E(ωn/2k​)

F(ωnk+n/2)=O(ωn/2k)−ωnk×E(ωn/2k)F(ω^{k+n/2}_n)=O(ω^{k}_{n/2})-ω^k_n\times E(ω^{k}_{n/2}) F(ωnk+n/2​)=O(ωn/2k​)−ωnk​×E(ωn/2k​)

不断向上合并答案，最终即可得到 f(x),g(x)f(x),g(x)f(x),g(x) 的点值序列，对应相乘即可。

3. IDFT 的实现：

即 DFT 的逆运算，将求出的 点值表示 转换为 系数表示。

可以通过 单位根求逆 或者 构造法 证明：把 DFT 中的 ωniω^i_nωni​ 换成 ωn−1ω_n^{-1}ωn−1​，做完之后除以 n 即可实现 IDFT。

4. 位逆序置换优化：

我们注意到原版 FFT 进行了大量数组复制操作，加上是递归版本自带的小常数，很容易被卡。

通过分析递归底层的 fif_ifi​ 数组，我们惊讶地发现每个位置编号对应的正是原位置编号的 二进制翻转。

利用数位 DP 的思想，我们设 RxR_xRx​ 为 二进制数 x 进行二进制翻转后的大小，且长度是固定的 N=2nN=2^nN=2n （就像我们预先设计的那样）。那么得到如下转移方案：

  • 由于从小到大求解，我们已经得知了 Rx>>1R_{x>>1}Rx>>1​ 的大小，x>>1 此时代表着 二进制 x 除去个位后的值，而移位的同时导致最高位多了一个前导 0 。为了消除该影响，我们只需要将其再次移位即可。

  • 对于个位，若其为 1 我们要将其翻转至最高位，直接或上 2n−1=N/22^{n-1} = N/22n−1=N/2 即可。

综上，转移方程为：

R(x)=R(x/2)>>1 ∣ (x mod 2)∗2n−1R(x)=R(x/2)>>1 ~|~ (x~\text{mod}~2)*2^{n-1} R(x)=R(x/2)>>1 ∣ (x mod 2)∗2n−1

同时我们也得到了非迭代版的FFT：

struct cp{

	{return cp(x+B.x,y+B.y);}
	cp operator - (cp const &B) const
	{return cp(x-B.x,y-B.y);}

	{return cp(x*B.x-y*B.y,x*B.y+y*B.x);}

int n,m,ro[maxn];

void fft(cp *f,int c) 
{
	for (int i=0;i<n;i++) if (i<ro[i]) swap(f[i],f[ro[i]]);  
	//直接得到递归至 1 的数组序列，非迭代向上转移 
	for (int h=2;h<=n;h<<=1)
	{
		cp w0( cos(2*pi/h), c*sin(2*pi/h) );
		for (int j=0;j<n;j+=h){
			cp buf(1,0);
			for (int i=j;i<(j+(h>>1));i++) {
				cp u=f[i]; cp t=buf*f[i+(h>>1)];
				f[i]=u+t; f[i+(h>>1)]=u-t;
				buf=buf*w0;
			}
		}
	}
}

我们之所以选择 NTT 而非 FFT ，是因为后者由于使用单位根的关系，精度多多少少受到影响，加上部分数论题目要求取模的限制，找一个膜意义下的 “单位根”替身 未尝不可。

肆. 多项式儿童套餐

仅仅知道如何求出两个多项式的卷积没啥用，对于大多题目只有知道如何转化才能利用所学的知识解题。

但学的内容又比较肤浅，既没有 多项式计数 也没有 多项式指数对数计算，多项式快速幂 这些高阶算法。算不上全家桶，就当是儿童套餐吧。

一. 生成函数入门

对于一个序列 aia_iai​ ，它的生成函数记做F(x)=∑ai∗xiF(x)=\sum a_i*x^iF(x)=∑ai​∗xi，注意到 aia_iai​ 可以有限项，也可以无限项。

但不论是哪一种情况，如果我们能搞出生成函数的系数序列，我们也就推出了 aia_iai​ 的通项，这是确定的。

三. 例题：K 阶差分前缀和

题目大意： 给定一序列 aia_iai​ 与 KKK ，求其 K 阶差分 or 前缀和序列。

解析：

我们对于 aia_iai​ 给出他的生成函数 G(x)=∑iai∗xiG(x)=\sum_i a_i*x^iG(x)=∑i​ai​∗xi ，并尝试构造一个函数 G(x)G(x)G(x)，使二者的卷积就是我们答案。

我们容易想到的是，对于 k=1k=1k=1 的前缀和，我们可以构造 G(x)=∑1⋅xiG(x)=\sum 1\cdot x^iG(x)=∑1⋅xi，由卷积定义我们可以得到 F(x)∗G(x)=∑xi⋅∑k≤iakF(x)*G(x)=\sum x^i\cdot \sum_{k\leq i} a_kF(x)∗G(x)=∑xi⋅∑k≤i​ak​，即其第 i 项对应了 aia_iai​ 的一阶前缀和。

同理，一阶差分要求前后两项相减，不难得到 G(x)=1−xG(x)=1-xG(x)=1−x 便可满足条件。

那么我们应该如何把它拓展到 k 阶呢？由于卷积的结合律，若要做 k 次前缀和，就相当于 F(x)∗G(x)kF(x)*G(x)^kF(x)∗G(x)k，也就是 G(x)=1(1−x)k=(1−x)−kG(x)=\frac{1}{(1-x)^k}=(1-x)^{-k}G(x)=(1−x)k1​=(1−x)−k。对于 kkk 阶差分，也同样可以得到G(x)=(1−x)kG(x)=(1-x)^kG(x)=(1−x)k。

而两者的处理方式却又微妙的差别：

  1. 对于差分的情况 运用二项式定理，其系数表示为 bi=(−1)i⋅(ki)b_i=(-1)^i\cdot \binom{k}{i}bi​=(−1)i⋅(ik​)，根据 n 的范围完全可以线性递推。

  2. 对于前缀和的情况，运用广义二项式定理，其系数表示为

bi=(−1)i⋅−ki‾i !=(−1)i∗(−k)(−k−1)...(−k−n+1)i !b_i=(-1)^i\cdot \frac{-k^{\underline i}}{i~!}=(-1)^i*\frac{(-k)(-k-1)...(-k-n+1)}{i~!} bi​=(−1)i⋅i !−ki​​=(−1)i∗i !(−k)(−k−1)...(−k−n+1)​
