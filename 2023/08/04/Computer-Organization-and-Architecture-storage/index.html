и®Ўз®—жңәз»„жҲҗеҺҹзҗҶпјҡеӯҳеӮЁзі»з»ҹ

2023-08-04 дё“дёҡеӯҰд№  2.5k еӯ—
и®Ўз®—жңәз»„жҲҗеҺҹзҗҶ

иҙөжҳҜиҙөпјҢдҪҶ Surface дәҢеҗҲдёҖи§ҰеұҸиҝҳжҳҜеӨӘйҰҷдәҶгҖӮ

дј з»ҹиҠҜзүҮжҖ§иғҪж”№иҝӣ

  • е…Ғи®ёйҮҚеӨҚеӯҳеҸ–иЎҢзј“еҶІеҢәпјҢиҖҢж— йңҖеўһеҠ еҸҰеӨ–зҡ„иЎҢеӯҳеҸ–ж—¶й—ҙ

    • еҰӮеўһеҠ  еҝ«йЎөжЁЎејҸ пјҢйҮҮз”Ё EDRAM зӯүгҖӮеҰӮжһңдёӢж¬Ўи®ҝй—®зҡ„иҝҳжҳҜиҜҘиЎҢпјҢеҲҷд»ҺдёҖдёӘе°Ҹе®№йҮҸзҡ„ SRAM дёӯеҸ–еҮә

  • еўһеҠ ж—¶й’ҹдҝЎеҸ·пјҢдҪҝеӯҳеӮЁеҷЁе’ҢеӨ„зҗҶеҷЁ дҝқжҢҒеҗҢжӯҘ

    • SDRAMпјҢд»Ҙж—¶й’ҹзҡ„дёҠеҚҮжІҝдј йҖҒж•°жҚ®
    • дёәдәҶеҠ йҖҹпјҢеңЁж—¶й’ҹзҡ„дёҠеҚҮе’ҢдёӢйҷҚжІҝйғҪдј иҫ“ж•°жҚ® DDR

еҗҢжӯҘDRAMпјҲSRAMпјү& DDR

  • е°ҶдёҺ CPU ејӮжӯҘе·ҘдҪңжЁЎејҸж”№дёәеҗҢжӯҘе·ҘдҪңжЁЎејҸпјҢеҚіжӢҘжңүиҮӘе·ұзҡ„иЎҢең°еқҖгҖҒеҲ—ең°еқҖиҜ‘з Ғпјӣ
  • еӨҡеӯҳеӮЁдҪ“пјҲBankпјүй…ҚзҪ®пјӣ
  • CPUеҸ‘еҮә ACTе‘Ҫд»Ө еҗҺпјҢеҶҚеҸ‘иҜ»жҲ–еҶҷе‘Ҫд»ӨиҝӣиЎҢи®ҝй—®пјӣ
  • зј–зЁӢжЁЎејҸеҜ„еӯҳеҷЁпјҢи®ҫзҪ®зӘҒеҸ‘й•ҝеәҰпјҲ1,2,4,8 зӯүпјүпјҢеҚіең°еқҖеҸҜд»ҘиҮӘеўһпјӣ


  • DIMM (Dual In-line Memory Module) еҸҢиҫ№жҺҘи§ҰеҶ…еӯҳжЁЎеқ—пјҢеҸҜйҖҡиҝҮеӯ—жү©еұ•гҖҒдҪҚжү©еұ•зҡ„ж–№ејҸеўһеҠ е®№йҮҸгҖӮ

  • SDRAM жҳҜйҖҡиҝҮдёӨйқўжҺҘи§Ұж§ҪгҖҒжҜҸдёҖдёӘиҠҜзүҮдёҠеҢ…жӢ¬иӢҘе№Іе№¶иЎҢе·ҘдҪңзҡ„ BANKпјҢиҖҢжҜҸдёӘ BANK жңүиӢҘе№І PLATE.


  • DDR еӯҳеӮЁеҷЁзҡ„иҜ»еҸ–еҺҹзҗҶпјҢж ёеҝғжҖқжғіжҳҜеўһеҠ е№¶иЎҢжҖ§пјҡ


DRAM иҜ»еҶҷзҡ„жӯЈзЎ®жҖ§ж ЎйӘҢ

дёәжҸҗй«ҳ DRAM еӯҳеӮЁеҷЁиҜ»еҶҷж“ҚдҪңзҡ„жӯЈзЎ®жҖ§дёҺеҸҜйқ жҖ§пјҲеҰӮиҲӘеӨ©гҖҒж•°жҚ®еә“пјүпјҢ еңЁеҶҷе…Ҙ m дҪҚж•°жҚ®зҡ„еҗҢж—¶иҝҳйңҖеҶҷе…Ҙ k дҪҚйҷ„еҠ дҪҚпјҢеҚіеҶҷе…Ҙз»ҸиҝҮзә й”ҷз ҒпјҲECCпјүзј–з Ғзҡ„ж•°гҖӮ


## еӨҡжЁЎеқ—еӯҳеӮЁеҷЁ

з”ұиӢҘе№Ізҡ„жЁЎз»„жүҖз»„жҲҗзҡ„дё»еӯҳжҳҜзәҝжҖ§зј–еқҖзҡ„пјҢиҖҢең°еқҖзҡ„зј–жҺ’жңүдёӨз§ҚжЁЎејҸпјҡйЎәеәҸдёҺдәӨеҸүпјҲеӨҡйҖҡйҒ“пјүжЁЎејҸгҖӮ


1. йЎәеәҸеӯҳеӮЁпјҲеӣһйЎҫпјү

  • йЎәеәҸеӯҳеӮЁе°ұжҳҜең°еқҖеңЁзү©зҗҶдёҠжҳҜиҝһз»ӯзҡ„пјҢи®ҝй—®ең°еқҖжҢүйЎәеәҸеҲ’еҲҶжҲҗиӢҘе№ІжЁЎеқ—пјӣ
  • зү№зӮ№жҳҜдёІиЎҢи®ҝй—®гҖҒеҗ„жЁЎеқ—зӢ¬з«Ӣе·ҘдҪңпјӣеўһеҠ жЁЎеқ—еҸҜжү©е……еӯҳеӮЁеҷЁе®№йҮҸпјӣ
  • еҰӮдёӢеӣҫпјҢдҪҺ 3 дҪҚйҖүжӢ©еӯ—ең°еқҖпјҢй«ҳ 2 дҪҚйҖүжӢ©жЁЎеқ—пјӣ


2. дәӨеҸүж–№ејҸ

  • ең°еқҖз Ғзҡ„дҪҺдҪҚеӯ—ж®өйҖүжӢ©дёҚеҗҢзҡ„жЁЎеқ—пјҢиҖҢй«ҳдҪҚеӯ—ж®өзЎ®е®ҡзӣёеә”жЁЎеқ—еҶ…зҡ„еӯҳеӮЁеӯ—пјӣ
  • иҝһз»ӯең°еқҖеҲҶеёғеңЁдёҚеҗҢзҡ„жЁЎеқ—дёӯпјҢеҗҢдёҖжЁЎеқ—зҡ„ең°еқҖдёҚиҝһз»ӯпјӣ
  • еҜ№дәҺиҝһз»ӯеӯ—зҡ„дј йҖҒеҸҜе®һзҺ°еӨҡжЁЎеқ—зҡ„жөҒж°ҙејҸе№¶иЎҢеӯҳеҸ–пјҢжҸҗй«ҳеӯҳеӮЁеҷЁеёҰе®ҪгҖӮ
  • еҰӮдёӢеӣҫпјҢдҪҺ2дҪҚйҖүжӢ©жЁЎеқ—пјҢиҖҢй«ҳ3дҪҚйҖүжӢ©жЁЎеқ—дёӯзҡ„еӯ—пјӣ


3. еӨҡжЁЎеқ—зҡ„еҹәжң¬з»“жһ„

  • дё»еӯҳиў«еҲҶжҲҗиӢҘе№ІзӢ¬з«Ӣзҡ„жЁЎеқ—пјҢиҖҢжҜҸдёӘжЁЎеқ—еқҮжңүиҮӘе·ұзӢ¬з«Ӣзҡ„иҜ»еҶҷжҺ§еҲ¶з”өи·ҜгҖҒең°еқҖе’Ңж•°жҚ®еҜ„еӯҳеҷЁпјҢ дёҺ CPU дј йҖҒдҝЎжҒҜпјӣ
  • дёҺйЎәеәҸеӯҳеӮЁдёҚеҗҢпјҢCPU еҸҜд»ҘеңЁдёҖдёӘеӯҳеҸ–е‘ЁжңҹеҶ…еҗҢж—¶и®ҝй—®еӣӣдёӘжЁЎеқ—пјҢеӯҳеӮЁеҷЁжҺ§еҲ¶ еҲҶж—¶дҪҝз”Ё ж•°жҚ®жҖ»зәҝиҝӣиЎҢдҝЎжҒҜзҡ„дј йҖ’пјӣ


4. е®ҡйҮҸи®Ўз®—

еҒҮи®ҫжЁЎеқ—еӯ—й•ҝзӯүдәҺж•°жҚ®жҖ»зәҝе®ҪеәҰпјҢдё”жЁЎеқ—еӯҳеҸ–дёҖдёӘеӯ—зҡ„еӯҳеӮЁе‘Ёжңҹдёә TпјҢжҖ»зәҝдј йҖҒе‘Ёжңҹдёә П„П„П„пјҢеӯҳеӮЁеҷЁзҡ„дәӨеҸүжЁЎеқ—ж•°дёә mпјҢеҲҷжңүпјҡ

  • еҜ№дәҺйЎәеәҸеӯҳеӮЁпјҢиҝһз»ӯиҜ»еҸ– m дёӘеӯ—йңҖиҰҒпјҡT0=mTT_0 = mTT0вҖӢ=mTпјӣ
  • еҜ№дәҺдәӨеҸүеӯҳеӮЁпјҢдёәдәҶдҝқиҜҒжөҒж°ҙзәҝжӯЈеёёе·ҘдҪңпјҢеә”ж»Ўи¶і m=T/П„m=T/П„m=T/П„пјӣжӯӨж—¶иҝһз»ӯиҜ»еҸ– m дёӘеӯ—жүҖйңҖж—¶й—ҙпјҡT1=T+(mвҲ’1)П„T_1=T+(m-1) П„T1вҖӢ=T+(mвҲ’1)П„пјӣ


5. дәӨеҸүеӯҳеӮЁеҷЁ

  • еӣһйЎҫпјҡеӯҳеӮЁеҷЁиҝӣиЎҢдёҖж¬ЎиҜ»еҶҷж“ҚдҪңжүҖйңҖзҡ„ж—¶й—ҙ з§°дёәеӯҳеӮЁеҷЁзҡ„и®ҝй—®ж—¶й—ҙпјҲжҲ–иҜ»еҶҷж—¶й—ҙпјүпјӣиҖҢиҝһз»ӯеҗҜеҠЁдёӨж¬ЎзӢ¬з«Ӣзҡ„вҖңиҜ»вҖқжҲ–вҖңеҶҷвҖқж“ҚдҪңпјҲеҰӮиҝһз»ӯзҡ„дёӨж¬ЎвҖңиҜ»вҖқж“ҚдҪңпјүжүҖйңҖзҡ„жңҖзҹӯж—¶й—ҙпјҢз§°дёәеӯҳеҸ–е‘Ёжңҹпјӣ

    дёӯй—ҙеҸҜиғҪиҰҒеҒҡеҲ·ж–°пјҲйҮҚеҶҷпјүд№Ӣзұ»зҡ„ж“ҚдҪңпјҢдҪҶжҳҜж•°жҚ®иҝҷж—¶еҖҷе·Із»ҸеҸ–еҲ°дәҶпјҢжҲ‘жғіз«ӢеҲ»еҸ–дёӢдёҖдёӘж•°жҚ®гҖӮжүҖд»Ҙеј•е…ҘдәӨеҸүеӯҳеӮЁеҷЁе№¶иЎҢиҜ»еҸ–е…¶д»–жЁЎеқ—зҡ„ж•°жҚ®пјҢд»ҺиҖҢеўһеӨ§еёҰе®Ҫпјӣ

  • д»ҘдәҢжЁЎеқ—дәӨеҸүеӯҳеӮЁеҷЁдёәдҫӢпјҡ

    • е…ұжңү 8 дёӘ 2MB еӯҳеӮЁдҪ“пјҢең°еқҖй«ҳ 3 дҪҚең°еқҖз”ЁдәҺеӯҳеӮЁдҪ“йҖүжӢ©(еӯ—жү©еұ•)пјӣ
    • A20~A3 еқ—еҶ…йҖүжӢ©ең°еқҖпјҢдҫқз„¶йҮҮз”ЁеҲҶж—¶еӨҚз”Ёзҡ„ж–№ејҸдј иҫ“иЎҢең°еқҖдёҺеҲ—ең°еқҖпјӣ
    • A2 жЁЎеқ—йҖүжӢ©пјҢдҪҝеҫ—еҒ¶ең°еқҖеңЁжЁЎеқ—0гҖҒеҘҮең°еқҖеңЁжЁЎеқ—1пјӣ


й«ҳйҖҹзј“еӯҳпјҲCacheпјү

  • иғҢжҷҜпјҡCache жҳҜдёҖз§Қй«ҳйҖҹзј“еҶІеӯҳеӮЁеҷЁпјҢжҳҜдёәдәҶи§ЈеҶі CPU йҖҹеәҰеӨӘеҝ«иҖҢдё»еӯҳзҡ„йҖҹеәҰеӨӘжӢүд№Ӣй—ҙзҡ„зҹӣзӣҫиҖҢжҸҗеҮәзҡ„жҠҖжңҜпјӣ

  • Cache дёҖиҲ¬з”ұ SRAM з»„жҲҗпјҢеӨ§е°ҸдёәеҮ зҷҫ KBпјҢиў«еөҢе…ҘеҲ° CPU дёӯеҒҡжҢҮд»ӨдёҺж•°жҚ®зҡ„зј“еӯҳпјӣ

  • дёҖиҲ¬дјҡеҢәеҲҶдёәеӨҡжЎЈпјҢеҰӮ L1 cache, L2 cache зӯүпјҢеҪўжҲҗйҮ‘еӯ—еЎ”з»“жһ„пјӣеҲ©з”ЁеұҖйғЁжҖ§еҺҹзҗҶпјҢеңЁCacheдёӯе®ҢжҲҗеӨ§еӨҡж•°и®ҝй—®пјҢд»ҺиҖҢзј©зҹӯе№іеқҮи®ҝй—®ж—¶й—ҙпјӣ

  • еј•е…ҘеӨ–еӯҳпјҲеҰӮзЎ¬зӣҳпјүпјҢжҲ‘д»¬и§ЈеҶідәҶеҶ…еӯҳе®№йҮҸе°Ҹзҡ„й—®йўҳпјӣеј•е…Ҙ cacheпјҢжҲ‘д»¬и§ЈеҶідәҶеҶ…еӯҳйҖҹеәҰж…ўзҡ„й—®йўҳпјҢе®һзҺ°дәҶеҜ№еҶ…еӯҳзҡ„жҠҪиұЎдёҺз»ҹдёҖпјӣ

  • Cache е…Ёз”ұзЎ¬д»¶и°ғеәҰпјҢеҜ№з”ЁжҲ·йҖҸжҳҺпјӣ


0. еұҖйғЁжҖ§еҺҹзҗҶ

дёҖдёӘзј–еҶҷиүҜеҘҪзҡ„зЁӢеәҸеёёеёёе…·жңүиүҜеҘҪзҡ„еұҖйғЁжҖ§гҖӮе®ғеҖҫеҗ‘дәҺйҮҚеӨҚи®ҝй—®зӣёеҗҢзҡ„ж•°жҚ®йЎ№йӣҶеҗҲпјҲж—¶й—ҙеұҖйғЁжҖ§пјүпјҢжҲ–иҖ…еҖҫеҗ‘дәҺи®ҝй—®йӮ»иҝ‘зҡ„ж•°жҚ®йЎ№йӣҶеҗҲпјҲз©әй—ҙеұҖйғЁжҖ§пјүгҖӮ

дҫӢпјҡC иҜӯиЁҖж•°з»„жҢүиЎҢдјҳе…Ҳж–№ејҸеӯҳеӮЁеңЁеӯҳеӮЁеҷЁдёӯпјҢеӣ жӯӨе·Ұдҫ§зҡ„д»Јз ҒжӣҙеҘҪзҡ„еҲ©з”ЁдәҶеұҖйғЁжҖ§еҺҹзҗҶпјҲдҪҶе®һйҷ…дёҠзј–иҜ‘еҷЁдјҡеҒҡдјҳеҢ–пјүпјӣ


2. Cache еҹәжң¬еҺҹзҗҶ

  • Cache е‘ҪдёӯпјҡеҸҜд»ҘеңЁ Cache дёӯжүҫеҲ°иҰҒи®ҝй—®ең°еқҖзҡ„еҜ№еә”еқ—е„ҝпјӣ
  • CacheзјәеӨұпјҡиҜҘеұӮзҡ„ Cache жүҫдёҚеҲ°пјҢеҝ…йЎ»еҺ»дёӢдёҖеұӮжүҫпјӣ


1. Cache жҺ§еҲ¶йғЁд»¶

  • ең°еқҖеҸҳжҚўпјҡеҰӮдҪ•е°Ҷдё»еӯҳдёӯзҡ„ең°еқҖжҳ е°„жҲҗ cache дёӯзҡ„ең°еқҖ

  • жӣҝжҚўз®—жі•пјҡеңЁcacheдёҚе‘Ҫдёӯж—¶еҰӮдҪ•жӣҝжҚўcacheдёӯзҡ„ еҶ…е®№

  • жӣҙж–°з®—жі•пјҡеҰӮдҪ•дҝқжҢҒдё»еӯҳдёҺcacheзҡ„дёҖиҮҙжҖ§


1.1 еҶ…е®№еҸҜеҜ»еқҖеӮЁеӯҳеҷЁCAM

  • CAM жҳҜдёҖз§ҚеҶ…е®№еҜ»еқҖзҡ„еӯҳеӮЁеҷЁгҖӮе°ҶдёҖдёӘиҫ“е…ҘдёҺеӯҳеӮЁзҡ„жүҖжңүж•°жҚ®иҝӣиЎҢжҜ”иҫғпјҢиӢҘеӯҳеңЁеҢ№й…ҚпјҢеҲҷиҫ“еҮәиҜҘж•°жҚ®йЎ№еҜ№еә”зҡ„еҢ№й…ҚдҝЎжҒҜгҖӮеҗҢж—¶д№ҹиғҪжҢүең°еқҖиҝӣиЎҢиҜ»е’ҢеҶҷгҖӮ

  • дёәдәҶжҠҠдё»еӯҳеқ—ж”ҫеҲ° Cache дёӯпјҢеҝ…йЎ»еә”з”Ёжҹҗз§Қж–№жі•жҠҠдё»еӯҳең°еқҖеҸҳжҚўжҲҗ Cache дёӯзҡ„ең°еқҖпјҢз§°дҪңең°еқҖжҳ е°„гҖӮ

    еҜ№дәҺжҜҸдёҖдёӘдё»еӯҳдёҠзҡ„ең°еқҖ 2n2^n2n пјҢе°Ҷе…¶иЎЁзӨәдёәеқ—еӨ§е°ҸпјҲ2b2^b2bпјүд»ҘеҸҠеқ—зј–еҸ·пјҲ2m2^m2mпјүдёӨйғЁеҲҶпјӣеҜ№дәҺ cacheпјҢжҲ‘д»¬дҝқжҢҒеқ—еӨ§е°ҸдёҚеҸҳпјҢиҖҢеқ—е„ҝж•°пјҲ2c2^c2cпјүиҝңе°ҸдәҺ 2m2^m2mгҖӮ

  • ең°еқҖжҳ е°„ж–№ејҸ

    • е…ЁзӣёиҒ”жҳ е°„пјҲ Fully Associative Mappingпјү

    • зӣҙжҺҘжҳ е°„пјҲDirect Mappingпјү

    • V-way з»„зӣёиҒ”жҳ е°„пјҲSet Associative Mapping пјү


1.1.1 е…Ёзӣёиҝһжҳ е°„
  • дё»еӯҳзҡ„д»»ж„Ҹеқ—е„ҝеҸҜжҳ е°„иҮі cache зҡ„д»»ж„Ҹеқ—е„ҝпјӣжҜҸж¬ЎеҲ©з”Ё CAM жҹҘиҜўеҚіеҸҜпјӣ
  • дјҳзӮ№пјҡCacheзҡ„еӯҳеӮЁз©әй—ҙеҲ©з”ЁзҺҮй«ҳпјҢе‘ҪдёӯзҺҮиҫғй«ҳпјӣ
  • зјәзӮ№пјҡе…ЁиҝһжҺҘжүҖйңҖиҰҒзҡ„ CAM еӯҳеӮЁеҷЁзәҝи·ҜеӨҚжқӮпјӣ
  • еӨ§иҮҙиҝҮзЁӢе°ұжҳҜе…ҲжҠҠеүҚ 19 дҪҚзҡ„еқ—е„ҝйҖүең°еқҖдҪңдёә tag еӯҳеӮЁиҮі CAMпјҢеҢ№й…Қзҡ„ж—¶еҖҷжҹҘиҜў CAM дёӯзҡ„жүҖжңүж ҮзӯҫпјҢж №жҚ®еҒҸз§»йҮҸиҜ»еҮәеҜ№еә”зҡ„еӯ—пјӣ


1.1.2 зӣҙжҺҘжҳ е°„
  • ж №жҚ®дё»еӯҳеқ—е„ҝзҡ„жңҖеҗҺиӢҘе№ІдҪҚпјҢе”ҜдёҖзЎ®е®ҡдё»еӯҳзҡ„зү№е®ҡиЎҢең°еқҖпјҲзұ»жҜ”еҸ–жЁЎе“ҲеёҢпјүпјӣжҳҜеӨҡеҜ№дёҖзҡ„жҳ е°„е…ізі»пјӣ
  • зү№зӮ№пјҡе®һзҺ°з®ҖеҚ•пјҢжҳ“дә§з”ҹеҶІзӘҒзјәеӨұпјӣ


1.1.3 v-way з»„зӣёиҒ”жҳ е°„
  • з»“еҗҲеүҚдёӨз§Қж–№ејҸпјҡе°Ҷ Cache еҲҶжҲҗ u з»„пјҢжҜҸз»„ v иЎҢпјҢдё»еӯҳеқ—еӯҳж”ҫеҲ°е“ӘдёӘз»„жҳҜеӣәе®ҡзҡ„пјҢжҹҘиҜўз»„дёӯе…ЁйғЁжҳ е°„зӣҙиҮіе‘Ҫдёӯorе…ЁйғЁзјәеӨұпјӣ


1.2 CacheеҶІзӘҒзјәеӨұе’Ңе®№йҮҸзјәеӨұ

иӢҘеҸ‘з”ҹеҶІзӘҒпјҢеҲҷйңҖиҰҒз”Ёж–°зҡ„ж•°жҚ®еқ—жӣҝжҚўж—§зҡ„ cache еқ—е„ҝпјӣ

еҸҜйҮҮеҸ–д»ҘдёӢзӯ–з•Ҙпјҡ

  • йҡҸжңәжӣҝжҚўжі•пјҲRANDпјүпјҡзЎ¬д»¶йҡҸжңәйҖүжӢ©дёҖдёӘ Cache ж•°жҚ®еқ—жӣҝжҚўеҮәеҺ»пјӣ
  • е…Ҳиҝӣе…ҲеҮәпјҲFIFOпјүпјҡжҠҠжңҖд№…иҝңзҡ„и°ғе…Ҙ cache зҡ„ж•°жҚ®еқ—дҪңдёәиў«жӣҝжҚўеҜ№иұЎпјӣ
  • жңҖе°‘дҪҝз”ЁпјҲLFUпјүпјҡе°ҶжңҖиҝ‘иў«и®ҝй—®ж¬Ўж•°жңҖе°‘зҡ„ж•°жҚ®еқ—жҚўеҮәпјҢйңҖдёәжҜҸдёӘ Cache иЎҢи®ҫзҪ®и®ҝй—®и®Ўж•°еҷЁпјӣ
  • жңҖд№…жңӘдҪҝз”ЁпјҲLRUпјүпјҡе°Ҷиҝ‘жңҹеҶ…жңҖй•ҝж—¶й—ҙжңӘиў«и®ҝй—®иҝҮзҡ„ж•°жҚ®еқ—жҚўеҮәгҖӮйңҖиҰҒдёәжҜҸдёӘCacheиЎҢи®ҫзҪ®и®Ўж—¶еҷЁпјӣ

1.3 Cache еҶҷж“ҚдҪңеӨ„зҗҶ

  • ж•°жҚ®еңЁеӯҳеӮЁеұӮж¬ЎдёӯеӯҳеңЁеӨҡд»ҪжӢ·иҙқпјҢеҰӮдҪ•дҝқиҜҒж•°жҚ®еҶҷж“ҚдҪңеӨ„зҗҶ cache е’ҢеӯҳеӮЁеҷЁдёӯдёҖиҮҙжҖ§гҖӮ

  • еҶҷе‘ҪдёӯпјҲwrite hitпјүзҡ„жғ…еҶөдёӢпјҡ

    • зӣҙеҶҷжі•пјҲWrite Throughпјүпјҡз«ӢеҚіе°Ҷ w зҡ„ cache еқ—е„ҝеҶҷеӣһдҪҺдёҖеұӮдёӯпјҢиҷҪз„¶з®ҖеҚ• дҪҶдјҡеҚ з”ЁжҖ»зәҝжөҒйҮҸпјӣ
    • еҶҷеӣһжі•пјҲWrite BackпјүпјҡеҸӘеҶҷе…Ҙ CacheпјҢеҸӘжңүжӣҝжҚўз®—жі•иҰҒй©ұйҖҗеҪ“еүҚеқ—е„ҝж—¶пјҢжүҚе°Ҷе…¶еҶҷеӣһдҪҺдёҖеұӮдёӯпјӣиҷҪз„¶иғҪеҮҸе°‘жөҒйҮҸ дҪҶйңҖиҰҒйўқеӨ–еўһеҠ дҝ®ж”№дҪҚпјҲеҚіжҮ’ж Үи®°пјүпјӣ

  • еҶҷдёҚе‘ҪдёӯпјҲwrite missпјүзҡ„жғ…еҶөдёӢпјҡ

    • йқһеҶҷеҲҶй…ҚпјҲnon-write-allocateпјүпјҡзӣҙжҺҘеҶҷеңЁдҪҺдёҖеұӮдёӯпјҢдёҚдјҡжӣҙж–° cacheпјҢжҗӯй…ҚзӣҙеҶҷжі•пјӣ
    • еҶҷеҲҶй…ҚпјҲwrite-allocateпјүпјҡе…ҲжҠҠж–°зҡ„зҡ„ж•°жҚ®еҶҷе…Ҙ cacheпјҢжҗӯй…ҚеҶҷеӣһжі•жӣҙж–°дҪҺдёҖеұӮзҡ„еқ—е„ҝгҖӮ

  • дёҖиҲ¬жқҘиҜҙпјҢеңЁ cache д№Ӣй—ҙйҖҹеәҰеҝ«пјҢйҮҮз”ЁзӣҙеҶҷжі•пјӣиҖҢ cache дёҺдё»еӯҳд№Ӣй—ҙиҫғж…ўпјҢйҮҮз”ЁеҶҷеӣһжі•гҖӮ
