Policy Function Approximation

之前我们学了值函数近似，现在继续来学策略函数近似。本质上都是一回事——用神经网络去拟合一个连续函数，替代之前采用的策略表。

令 π(a∣s,θ)\pi(a|s,\theta)π(a∣s,θ) 在状态 s 下选择动作 a 的概率，其中 θ\thetaθ 为函数参数。

表格形式与函数形式会有哪些不同呢？

  • 首先，在如何定义最优策略上：表格形式的最优策略就是对应策略下的任一 state value 大于等于其他策略下的 state value；而函数形式需要用一个标量作为 metrics，最大化这个目标度量。

  • 第二，如何检索某状态下的策略：表格形式直接查表就好，现在还要把状态丢进模型里，算一下才能知道。

  • 第三：如何更新策略：表格形式下直接改表格中的元素就好了，而函数形式需要借助修改参数间接更新策略。

    但这样也有好处，更改策略具有一定泛化性。之前改状态-动作对的概率只能一个个改，现在为了更新某个状态下的某个动作去修改参数时，可能也会连带着影响到其他相关的状态-动作对。

1. 如何定义最优策略

第一种 metric 称为平均状态价值，or average value simply.

当 d(s)d(s)d(s) 与选取策略无关时，所以在求梯度的时候可以直接拎出来。至于如何选择初始的 d(s)d(s)d(s) 呢？最简单的就是均匀分布，每个状态出现概率一样；或者对于某些游戏来说，选择某种初始状态 s0s_0s0​ 可以保证价值最大，那我们初始就令 d(s0)=1d(s_0)=1d(s0​)=1，其他状态 d(s′)=0d(s')=0d(s′)=0 .

dTPπ=dTd^TP_\pi = d^T dTPπ​=dT

θt+1=θt+α∇θln⁡π(at∣st,θt)qπ(st,at)\theta_{t+1}=\theta_t+\alpha \nabla_\theta \ln \pi\left(a_t \mid s_t, \theta_t\right) q_\pi\left(s_t, a_t\right) θt+1​=θt​+α∇θ​lnπ(at​∣st​,θt​)qπ​(st​,at​)

Off-Policy 的 Actor-Critic

之前的课我们可知，由于目标函数 ∇θJ(θ)=ES∼d,A∼π[∗]\nabla_\theta J(\theta) = \mathbb{E}_{S \sim d, A \sim \pi}[*]∇θ​J(θ)=ES∼d,A∼π​[∗] 中动作采样是要服从策略 π\piπ 的，是我们的 Behavior Policy；同时， π\piπ 也是我们要更新的 Target Policy. 因此这些算法都是 On-Policy 的。

如果想将策略梯度算法转换成 Off-Policy，去复用之前得到的经验，也是可以的。只需要用一种 Importance Sampling 即可。

PPO

我们引入 KL 散度，限制 π\piπ 和 β\betaβ 之间的差异，得到式子：

JPPOβ(θ)=Jβ(θ)−βKL(θ,θ′)where Jβ(θ)=E(st,at)∼β[pπ(at∣st)pβ(at∣st)Aβ(st,at)]\begin{aligned} J_{PPO}^\beta ( \theta) &=J^{\beta} ( \theta)-\beta K L ( \theta, \theta^\prime ) \\ \mathrm{where}~ J^{\beta} ( \theta) &= \mathbb{E}_{( s_t, a_t ) \sim\beta} \left[ \frac{p_\pi ( a_t | s_t )} {p_{\beta} ( a_t | s_t )} A^{\beta} ( s_t, a_t ) \right] \end{aligned} JPPOβ​(θ)where Jβ(θ)​=Jβ(θ)−βKL(θ,θ′)=E(st​,at​)∼β​[pβ​(at​∣st​)pπ​(at​∣st​)​Aβ(st​,at​)]​

实际上，“两个策略 π\piπ 和 β\betaβ 不能相差太多” 并不是指参数层面差太多，而是这两个策略导致的行为分布应该是差不多的（即在同一个 state 下，计算出 action 对应的分布，计算这两个之间的 KL 散度）。

DPO

1. 为什么选择相对奖励偏好，而非绝对的奖励打分？

  • 人与人在分配绝对奖励方面没有校准，差别较大；

  • 相较之下，在两个回答中选择偏好的回答会容易得多。

p(yw≻yl∣x)=σ(r(x,yw)−r(x,yl))p ( y_{w} \succ y_{l} \mid x )=\sigma( r ( x, y_{w} )-r ( x, y_{l} ) ) p(yw​≻yl​∣x)=σ(r(x,yw​)−r(x,yl​))

3. RM有了，如何学习策略？

假设奖励模型 rϕr_{\phi}rϕ​ 有了，我们要继续训练策略模型 πθ\pi_{\theta}πθ​ 以实现更高的价值，从策略中采样 yyy，得到第一版目标函数：

max⁡πθEx∼D,y∼πθ(y∣x)[rϕ(x,y)]\max_{\pi_{\theta}} \mathbb{E}_{x \sim\mathcal{D}, y \sim\pi_{\theta} ( y | x )} \Big[ r_{\phi} ( x, y ) \Big] πθ​max​Ex∼D,y∼πθ​(y∣x)​[rϕ​(x,y)]

然而，一昧按照这个目标函数优化是无意义的，因为 rϕr_\phirϕ​ 只是在一个有限的数据集上训练的，只能在某些分布上给出指导，而在分布之外的数据无法泛化。

因此，我们得到了第二版目标函数，加上了 KL 散度的约束，使 πθ\pi_{\theta}πθ​ 取得高奖励的同时也能和原模型的策略 πref\pi_{\mathrm{ref}}πref​ 相近。

根据先前的经验，其实可以得到封闭最优策略的解：

π∗(y∣x)=1Z(x)πref(y∣x)exp⁡(1βr(x,y))with Z(x)=∑yπref(y∣x)exp⁡(1βr(x,y))\pi^{*} ( y \mid x )={\frac1 {Z ( x )}} \pi_{\mathrm{ref}} ( y \mid x ) \exp \left( {\frac1 {\beta}} r ( x, y ) \right) \\ \mathrm{with} ~ Z ( x )=\sum_y \pi_{\mathrm{ref}} ( y \mid x ) \exp \left( {\frac1 {\beta}} r ( x, y ) \right) π∗(y∣x)=Z(x)1​πref​(y∣x)exp(β1​r(x,y))with Z(x)=y∑​πref​(y∣x)exp(β1​r(x,y))

r(x,y)= βlog⁡π∗(y∣x)πref(y∣x)+βlog⁡Z(x)r ( x, y )=\ \beta\log \frac{\pi^{*} ( y \mid x )} {\pi_{\mathrm{r e f}} ( y \mid x )}+\beta\log Z ( x ) r(x,y)= βlogπref​(y∣x)π∗(y∣x)​+βlogZ(x)

在实践中，PPO 和 DPO 的选择取决于具体任务的需求：

  • PPO 更加适合那些有明确奖励信号且需要稳定学习过程的任务，如游戏AI或机器人控制。而且由于其广泛适用性和相对成熟的方法论，PPO 在许多标准强化学习基准中表现良好。
  • DPO 则在需要高度对齐人类价值观和偏好的应用中表现出色，如自然语言处理中的文本生成任务。它能够快速适应人类反馈，但可能在数据稀缺或偏差时表现不稳定。
